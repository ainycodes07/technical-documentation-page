Introduction

MinBrowser is a fast, minimal, and high-performance web browser designed for focus and simplicity.

It emphasizes speed and efficiency, stripping away unnecessary UI elements.

The browser is open-source and actively maintained by a community of contributors.

Its small footprint makes it ideal for developers and productivity-focused users.

MinBrowser provides features like built-in ad-blocking, reading mode, and custom search engines.

  • Lightweight
  • Privacy-first
  • Cross-platform

Installation

MinBrowser can be installed across Windows, macOS, and Linux distributions.

For Linux, Min is available via AppImage, DEB, and RPM formats.

On macOS, you can install Min via Homebrew or direct download.

On Windows, the installer package is available on the official GitHub releases page.

You can also build MinBrowser from source for maximum customization.

  • macOS: brew install --cask min
  • Linux: sudo dpkg -i min.deb
  • Windows: Run the installer
    • git clone https://github.com/minbrowser/min.git
Features

MinBrowser offers powerful features despite its minimal design.

Its ad-blocker removes intrusive ads and trackers automatically.

It includes Focus Mode for distraction-free browsing.

The browser has built-in search shortcuts and customizable engines.

It consumes significantly less memory than most modern browsers.

  • Automatic ad blocking
  • Privacy-focused design
  • Custom search shortcuts
  • Distraction-free reading
  • Low memory usage

Configuration

MinBrowser allows configuration via settings and JSON files.

You can modify search engines, privacy preferences, and shortcuts.

Advanced users can configure startup behavior and UI elements.

Settings are stored locally and synced across sessions.

Configuration examples are shown below:

  • "searchEngine": "DuckDuckGo"
  • "focusMode": true
  • "adBlock": true
